
# Carousel
<br>
위코드 1차 프로젝트를 계기로 라이브러리, 커스텀 훅 없이 구현한 버튼, 자동 슬라이드입니다.<br>
자세한 구현 과정 
[Velog]
(https://velog.io/@rxxdo/React-PJ.-%EB%9D%BC%EC%9D%B4%EB%B8%8C%EB%9F%AC%EB%A6%AC-%EC%97%86%EC%9D%B4-%EB%B2%84%ED%8A%BC-%EC%9E%90%EB%8F%99%EC%8A%AC%EB%9D%BC%EC%9D%B4%EB%93%9C-%EA%B5%AC%ED%98%84)
에 작성해두었습니다.
<br>
<br>
주요 기능은 다음과 같습니다.
- 버튼 클릭 시 슬라이드 이동.<br>
- 이미지 별 2.5초 정지 후 자동 슬라이드 진행.<br>
